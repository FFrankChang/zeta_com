Zeta ACI Website Design Direction
Overall Visual Style

The website uses a premium light-mode futuristic aesthetic with a strong focus on clean whitespace, soft translucency, subtle gradients, and refined 3D elements.

The overall feeling should be:

intelligent

minimal

elegant

trustworthy

soft but technological

futuristic without being dark or aggressive

This is not a typical dark AI style. It should feel like a high-end AI product brand in a bright white environment.

Core Design Language
1. Light premium interface

Use a white to very-light-gray background as the base.

Surfaces should feel soft, elevated, and slightly diffused.

Avoid hard contrast blocks or overly saturated colors.

The interface should feel airy, calm, and polished.

2. Frosted glass / translucent UI

Cards, panels, and buttons should use a glassmorphism-inspired treatment:

semi-transparent white surfaces

soft borders

backdrop blur

subtle inner glow or highlight

very soft shadows

Transparency should be restrained and elegant, not flashy.

3. Soft blue–purple brand gradients

Primary accent colors come from a pastel technology palette:

ice blue

pale cyan

lavender

soft violet

Gradients should be:

smooth

low-saturation

slightly luminous

used for key accents, buttons, 3D objects, and decorative motion trails

Avoid harsh neon, deep cyberpunk colors, or overly playful rainbow effects.

Signature Visual Element

A core visual motif is a stacked cube / block-like AI agent figure.

Characteristics:

made of simple rounded translucent cubes

slightly transparent or frosted

soft blue/purple glow

minimal “face” or light marks can be used to suggest intelligence

should feel like a modular AI agent building block

This element represents:

AI agents

modular composition

orchestration

workflow assembly

intelligent system components

It should appear across the site as a consistent branded 3D object.

3D Illustration Style

3D visuals should follow these rules:

soft-edged geometry

translucent or semi-frosted materials

subtle refraction-like feel

gentle bloom/glow

pastel blue/lavender gradients

clean white environment lighting

no dark background renders

no heavy realism; keep it stylized and product-brand oriented

Acceptable 3D motifs:

stacked cubes

flowing ribbon-like forms

abstract smooth loops

modular components assembling and separating

These 3D elements should feel like AI systems in motion, but still minimal and controlled.

Layout Style

The layout follows a modular card-based composition.

Principles

Large rounded rectangular panels

Clear spacing between cards

Strong grid alignment

Lots of breathing room

Content grouped into clean visual blocks

Card treatment

large border radius

subtle border line

soft elevation shadow

light translucent background

minimal internal clutter

The page should feel like a design system board / premium product showcase, not a dense enterprise dashboard.

Typography

Typography should be:

clean

modern

sans-serif

highly legible

minimal and confident

Usage

Large bold hero headlines

Short supporting copy

Sparse navigation text

Clear section titles in uppercase or structured label style where needed

The typography should support the visual system rather than compete with it.

Component Style
Buttons

Rounded pill or soft rounded rectangle shape

Gradient fill using pale blue to lavender

Slight gloss/highlight effect

Soft shadow

Minimal border

Smooth hover transition

Cards

Frosted white background

Light border

Rounded corners

Soft shadow

Clean internal spacing

Icons

Outline or lightly filled icons

Rounded, soft, friendly geometry

Consistent with the translucent / glass aesthetic

Can sit inside subtle rounded square containers

Navigation

Minimal top navigation

Clean spacing

Light visual weight

Should not dominate the hero section

Interaction and Motion Style

Motion should feel fluid, intelligent, and quiet.

Motion principles

smooth easing

slow-to-medium speed

no abrupt or bouncy animation

subtle parallax is acceptable

gentle hover lift and glow

soft fade / blur / translate transitions

Recommended motion motifs

cubes assembling and disassembling

light trails connecting blocks

flowing ribbon motion between states

continuous looped transitions suggesting orchestration and iteration

Motion should communicate:

modular AI workflows

agents collaborating

transformation of small units into larger systems

continuous intelligence loops

UX Style

The UX should feel:

premium and guided

simple to scan

low cognitive load

visual-first

brand-forward

product-story oriented

UX principles

clear hierarchy

short content blocks

obvious CTAs

strong hero section

modular sections that explain capability step by step

avoid heavy enterprise complexity on the landing page

This should feel more like a high-end AI product website than a traditional corporate site.

Implementation Notes for Front-End

When building the UI, prioritize:

white/light-only design

soft glass surfaces

rounded corners

low-contrast shadows

subtle gradients

premium spacing

3D branded decorative elements

consistent cube-based AI agent motif

Do not use:

dark mode styling

harsh black backgrounds

aggressive neon effects

dense dashboard-like layouts

cartoonish illustrations

overly playful motion

visually heavy borders or sharp corners

One-Line Design Summary

A premium light-mode AI brand interface built with frosted glass surfaces, soft blue–purple gradients, modular card layouts, and translucent cube-based agent visuals that communicate intelligence, orchestration, and elegant future-facing technology.
